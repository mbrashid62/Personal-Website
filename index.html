Temperature Reader

This was an academic team project for my Senior Design class. Despite being frustratingly difficult at times, my group and I had a blast making this.

The project consisted of three distinct components. An Arduino circuit board for sensing temperature, a Java Program to collect serialized data from the board, and a web application to display the data in real-time.

The latter two components was what I worked on. This involved importing the Java RXTX library to handle I/O from the Arduino and programming Swing components to perform the appropriate functions. The JS bit was a bit tricky to figure out. Luckily, I used the Morris JS chart library. Morris makes creating displays a piece of cake if data is organized nicely in JSON format. I was able to use some handy jQuery functions to parse incoming data from the Java application into JSON and have the JavaScript interpret it every second for the chart.

The feeling when I finally got it working...oh man. I am going to chase that feeling!

Sail

Sail is a personal side project of mine that allows users to explore the vast ocean of SoundCloud genres. The app prompts users to enter in their favorite genres and it will spit out some suggested tracks they might like.

I had began working on Sail to get ready for my internship by practicing strong software architecture strategies. I also love music, so I figured why not use the SoundCloud API to learn something new and build something awesome?

Utilizing the Knockout framework, I gained a solid grasp of how Models, Views, and (in Knockout's case) ViewModels work. I actually restarted this project twice because I had failed epically during my first two attempts. I am not ashamed to admit that however, failure is all apart of the process right?!

Duck Hunt

Developed in Java as a final project for my Software Design course, this project utilized effective software design practices I learned all semester. These techniques include separation of concerns, organized code, smart use of javadocs/uml diagrams, and more.

This is a a multi-threaded application that uses Java Swing components to handle all of its events.

We were asked to build this application in one weekend, which happened to be the weekend right before finals. I think the most valuable lesson I learned from this project was how to stay cool under pressure. Many of my peers panicked and didn't finish in time. Never lose your cool under tight deadlines!
